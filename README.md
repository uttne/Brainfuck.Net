# Brainfuck.Net
Brainfuck.Net is brainfuck compiler.
This is global tools for dotnet core.

## Install
```
$ dotnet tool install --global Brainfuck.Net
```

## How to use
```
$ brainfuck.net --code "+++++++++[>++++++++>+++++++++++>+++++<<<-]>.>++.+++++++..+++.>-.------------.<++++++++.--------.+++.------.--------.>+.#"
```
